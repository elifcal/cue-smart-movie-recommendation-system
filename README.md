## Frontend

Frontend tarafı React + Vite ile geliştirilmiştir. Kullanıcının yazdığı doğal dil girdisini backend'e gönderir ve dönen film önerilerini kart yapısında gösterir.

### Özellikler
- Doğal dil ile film arama
- Örnek arama önerileri
- Film kartları
- Why text gösterimi
- Duygu eğrisi görselleştirmesi
- Renk paleti gösterimi
- Geri bildirim butonları
- Mobil uyumlu arayüz

### Kullanılan Teknolojiler
- React
- Vite
- Axios

```bash
cd frontend
npm install
